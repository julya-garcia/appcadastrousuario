# App Cadastro de Usuário

Aplicativo desenvolvido no Android Studio com o objetivo de realizar o cadastro de usuários por meio de uma interface simples e prática.

## Descrição
O projeto foi criado para treinar o desenvolvimento de aplicativos Android utilizando Java e XML, permitindo o preenchimento de dados do usuário em campos de texto e a interação com um botão de salvar.

## Funcionalidades
- Inserção do nome do usuário
- Inserção do e-mail
- Botão para salvar os dados
- Interface simples e organizada

## Tecnologias utilizadas
- Java
- XML
- Android Studio
- Material Design

## Estrutura do projeto
O projeto é composto por:
- `CreateUser.java` → classe responsável pela tela de cadastro
- `create_user.xml` → layout da interface
- Componentes visuais com `TextInputLayout`, `TextInputEditText` e `Button`

## Objetivo do projeto
Este aplicativo foi desenvolvido com fins de aprendizagem, para praticar:
- criação de telas no Android Studio
- uso de componentes visuais
- ligação entre Java e XML
- organização básica de um projeto Android
